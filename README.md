Encode (Python → Donk):
python Donklt.py encode input.py
python Donklt.py encode input.py -o output.donk

Decode (Donk → Python):
python Donklt.py decode input.donk
python Donklt.py decode input.donk -o output.py

Run / Execute Donk file:
python Donklt.py run input.donk

Run with arguments:
python Donklt.py run input.donk arg1 arg2 arg3

Run with offset:
python Donklt.py run input.donk --offset 32
