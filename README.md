Encode (Python → Donk): </br>
python Donklt.py encode input.py </br>
python Donklt.py encode input.py -o output.donk </br>

Decode (Donk → Python): </br>
python Donklt.py decode input.donk </br>
python Donklt.py decode input.donk -o output.py </br>

Run / Execute Donk file: </br>
python Donklt.py run input.donk </br>

Run with arguments: </br>
python Donklt.py run input.donk arg1 arg2 arg3 </br>

Run with offset: </br>
python Donklt.py run input.donk --offset 32 </br>
